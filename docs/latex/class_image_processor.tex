\subsection{Detailed Description}
Images processing class

\begin{Desc}

\begin{DoxyItemize}
\item supports jpg, png and gif
\item Preserves png and gif transparency
\item rotate images
\item create image thumbnails on the fly
\item Can be used with direct url imageprocessor.php?src=
\item Can be used as a class in your own website/application code
\item 3 types of image resizing (stretch, fit, crop)
\item watermark image with easy watermark positioning (pixels or top,bottom,center,left,right)
\item \hyperlink{class_cache}{Cache} images for efficiency
\end{DoxyItemize}
